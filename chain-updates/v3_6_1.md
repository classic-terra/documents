# Terra Classic - Update v3.6.1 (v13_1)

**Release Date**: 7 Dec 2025  </br>

**Chain Update Date**: 18 Dec 2025 </br>

## Key Changes
The release can be found here: https://github.com/classic-terra/core/releases/tag/v3.6.1

**Legacy Contract Fix:**
As approved in proposal #12193, this release includes a fix for legacy contracts that could not utilize specific features like tax and exchange rate querying.  
Please see https://discourse.luncgoblins.com/t/proposal-fix-legacy-contracts-by-core-upgrade/154 for details.

## Implications / Required Actions
The update is expected to re-enable functionality for legacy contracts, and thus to cause currently locked liquidity in such contracts to become accessible again.  
Users of legacy contracts are advised to check their contracts after the upgrade to see if functionality has been restored.

This also affects Astroport pools that were using legacy contracts. Users of such pools should be able to trade and withdraw liquidity again after the upgrade.

No impact on existing dApps or node providers are expected beyond the standard upgrade process.
